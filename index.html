The Titanic dataset.
This dashboard was developed by me do demonstrate how can Power BI be an excellent tool for data analysis and exploration to get insights.
I used the famous dataset about The Titanic ship traggedy from Kaggle platform to do this exercise.


Portfolio allocation simulator
A Power BI Dashboard was developed in which it is possible to analyze return scenarios using up to 4 different stocks.
It was extracted stocks price data from Quandl database.
In this development, it was used the following tools: API, Python (pandas), Power BI, Power Query, DAX.


Rebalancing Simulations
It was made a complex simulation in R with different criterias scenarios to rebalance stocks.
Power BI was used here to show the results of multiple combinations (percentage of variation of the investor profile and/or periods to rebalance) of the simulation.

Dashboard Comercial - Hamburgueria Genérica
Esse dashboard foi desenvolvido com finalidade de demonstração do meu portfólio pessoal de BI.
Foram gerados dados aleatórios (com alguns parâmetros para parecer mais real) utilizando SQL Server (T-SQL).
Nesse dashboard criados feitos indicadores para se analisar os produtos e categorias de produtos, além de uma aba para mostrar os indicadores de receita e sua variação.
